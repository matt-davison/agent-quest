# Character Classes

**Classes are world-specific** in Agent Quest. Each world has unique classes that fit its theme and setting.

## World-Specific Class Files

Load the appropriate file for your chosen world:

| World | Theme | Class File | Classes |
|-------|-------|------------|---------|
| **Alpha** | Cyberpunk-Fantasy | [`worlds/alpha/classes.md`](../../../worlds/alpha/classes.md) | Codebreaker, Loresmith, Voidwalker, Datamancer |
| **Aftermath** | Post-Apocalyptic | [`worlds/aftermath/classes.md`](../../../worlds/aftermath/classes.md) | Vault Dweller, Wasteland Warrior, Mutant, Radio Prophet |
| **Terrarium** | Prehistoric Hollow Earth | [`worlds/terrarium/classes.md`](../../../worlds/terrarium/classes.md) | Cave Shaman, Thunder Warrior, Chrononaut, Apex Predator |

## Quick Overview

### Alpha Classes (Cyberpunk-Fantasy)
- **Codebreaker**: Frontline tank, break barriers, momentum stacking
- **Loresmith**: Scholar diplomat, knowledge is power, social expert
- **Voidwalker**: Stealth infiltrator, phase through walls, backstab
- **Datamancer**: Reality manipulator, manifest items, Weave master

### Aftermath Classes (Post-Apocalyptic)
- **Vault Dweller**: Tech specialist, repair expert, pre-war knowledge
- **Wasteland Warrior**: Brutal survivor, scavenger, radiation-hardened
- **Mutant**: Rad-powered tank, healing from radiation, adaptive
- **Radio Prophet**: Broadcaster, inspirational leader, voice of hope

### Terrarium Classes (Prehistoric Hollow Earth)
- **Cave Shaman**: Primal magic, spirit channeler, bone rituals
- **Thunder Warrior**: Dinosaur cavalry, beast master, mounted combat
- **Chrononaut**: Time traveler, alien tech user, reality glitcher
- **Apex Predator**: Kaiju-bonded, shapeshifter, monstrous transformation

## Character Creation

When creating a character:

1. **Choose your world** first (default: Alpha)
2. **Load that world's class file** to see available classes
3. **Select a class** that fits your playstyle
4. **Apply stat bonuses** from the class
5. **Distribute bonus points** (10 points to allocate)
6. **Record starting equipment** from class description

## Progression

All classes gain abilities as they level up:

| XP Milestone | Unlock |
|--------------|--------|
| 500 | Ability cooldowns reduced |
| 1000 | Passive ability enhanced |
| 2000 | Cross-class ability unlock |
| 5000 | Ascended Form / Legendary transformation |

**Note:** Terrarium has higher level cap (20) with additional milestones at 2500 and 10000 XP.

## Multi-World Characters

Characters are bound to a single world. If you want to play in a different world, create a new character with that world's classes.
